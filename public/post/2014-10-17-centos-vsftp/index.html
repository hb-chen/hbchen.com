Centos VSFTP配置

2014-10-17
系统
89 words 1 min read

安装

yum install vsftp

配置

#/etc/vsftpd/vsftpd.conf
#关闭匿名登录
#anonymous_enable=NO

user_list中的说明是userlist_deny
#userlist_enable=NO

FTP root
#local_root=/mnt/ftp

添加登录用户

添加用户
$ useradd Hobo
设置密码
$ passwd Hobo
$加入user_list
$ echo Hobo >> /etc/vsftpd/user_list

重启

$service vsftpd restart
